# cpp_practice

A collection of simple C++ practice programs.

## Overview

This repository contains several small C++ programs designed for practice and learning purposes. Each program is contained in its own `.cpp` source file, and where available, accompanying output screenshot files are provided.

## Programs included

Here is a list of the programs currently present in this repo:

| Program Name            | Description                                                                                                                                                                                                                                                            |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `auth.cpp`              | A simple authentication‑style program (e.g., entering username/password). See [source](https://github.com/MohammadAliMehri/cpp_practice/blob/master/auth.cpp) and screenshot `auth_output.png`.                                                                        |
| `days_of_a_week.cpp`    | A program that asks the user for a number (1–7) and prints the corresponding day of the week. See [source](https://github.com/MohammadAliMehri/cpp_practice/blob/master/days_of_a_week.cpp) and screenshot `days_of_a_week_output.png`.                                |
| `sen.cpp`               | (Likely a short exercise) — a small practice program. See [source](https://github.com/MohammadAliMehri/cpp_practice/blob/master/sen.cpp) and screenshot `sen_output.png`.                                                                                              |
| `simple_calculator.cpp` | A basic calculator program (addition/subtraction/multiplication/division) using C++. See [source](https://github.com/MohammadAliMehri/cpp_practice/blob/master/simple_calculator.cpp) and screenshot `simple_calculator_output.png`. Includes `simple_calculator.exe`. |
