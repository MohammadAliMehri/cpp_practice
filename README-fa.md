# cpp_practice

مجموعه‌ای از برنامه‌های ساده برای تمرین زبان ++C.

---

این مخزن شامل چندین برنامه‌ی کوچک به زبان ++C است که برای تمرین و یادگیری طراحی شده‌اند. هر برنامه در یک فایل مجزای `.cpp` قرار دارد و در صورت وجود، فایل تصویری از خروجی برنامه نیز ضمیمه شده است.

## برنامه‌های موجود

در اینجا فهرستی از برنامه‌های موجود در این مخزن ارائه شده است:

| نام برنامه              | توضیحات                                                                                                                                                                                                                                                     |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `auth.cpp`              | برنامه‌ای ساده برای احراز هویت (مثلاً ورود نام کاربری و رمز عبور). مشاهده [سورس برنامه](https://github.com/MohammadAliMehri/cpp_practice/blob/master/auth.cpp) و تصویر خروجی `auth_output.png`.                                                             |
| `days_of_a_week.cpp`    | برنامه‌ای که عددی بین ۱ تا ۷ از کاربر دریافت می‌کند و روز هفته‌ی مربوطه را نمایش می‌دهد. مشاهده [سورس برنامه](https://github.com/MohammadAliMehri/cpp_practice/blob/master/days_of_a_week.cpp) و تصویر خروجی `days_of_a_week_output.png`.                   |
| `sen.cpp`               | یک برنامه‌ی تمرینی کوچک. مشاهده [سورس برنامه](https://github.com/MohammadAliMehri/cpp_practice/blob/master/sen.cpp) و تصویر خروجی `sen_output.png`.                                                                        |
| `simple_calculator.cpp` | یک ماشین حساب ساده (عملیات جمع، تفریق، ضرب، تقسیم). مشاهده [سورس برنامه](https://github.com/MohammadAliMehri/cpp_practice/blob/master/simple_calculator.cpp) و تصویر خروجی `simple_calculator_output.png`.
